How to Use

1. First Time Setup

Before using TTS, you need to download the voice models:

  • Click Resources → tab Download
  • Download Piper Voices (Low · CPU, ~848MB) — works on any machine
  • Download F5-TTS Model (Medium · GPU, ~1.3GB) — requires NVIDIA GPU
  • Download OmniVoice Model (High · GPU, ~2.3GB) — requires NVIDIA GPU 6GB+
  • Files are saved locally in the models/ folder — download once, use forever

2. Load Models

  • Piper (Low) — loads automatically, no action needed
  • F5-TTS (Medium) — click Resources → tab Load Models → Load F5
  • OmniVoice (High) — same tab → Load OmniVoice
  • Loading takes 10–60 seconds depending on your GPU

3. Generate Speech

  • Type or paste your text (max 5000 characters)
  • Select Low / Medium / High quality tier
  • Click Preview to hear a sample, or Generate for full audio
  • Long text is split into segments — you can replay or regenerate each one
  • Use Merge & Download All to combine all segments into one file

4. Features

  • Dict — custom pronunciation for acronyms and foreign words
  • Pause — adjust silence after punctuation, add [1s] markers
  • History — view and replay past generations
  • Advanced — speed, pitch, volume sliders
  • Normalize — auto-clean text and normalize volume
  • Voice Lab — clone a new voice from reference audio (F5 / OmniVoice)

5. Quality Tiers

Tier Engine Speed GPU?
Low Piper Fast No
Medium F5-TTS Medium Yes (4GB+)
High OmniVoice Slow Yes (6GB+)

Tip: Start with Low (Piper) — it works on any machine and is the fastest. Switch to Medium or High for better voice quality if you have a GPU.

6. Keyboard Shortcuts

Shortcut Action
Space Play / Pause audio
R Retry first failed segment
Ctrl + Enter Start generation

Shortcuts work when not typing in text fields.
